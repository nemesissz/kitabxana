🔧 Vergi API - Frontend Developer Demo

Base URL: https://api.muhasibatjurnal.az

Bu demo səhifə frontend developerlərin API ilə necə işləyəcəyini göstərir.

🔑 Authentication Token

API çağırışları üçün login olduqdan sonra aldığınız JWT token-i istifadə edin:

🔐 Authentication (Giriş/Qeydiyyat)

POST /auth/register - Qeydiyyat 
// Request body
{
    "email": "user@example.com",
    "password": "123456"
}
POST /auth/login - Giriş 
// Response
{
    "status": "success",
    "data": {
        "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
        "user": {
            "id": 1,
            "email": "user@example.com",
            "role": 1
        }
    }
}
